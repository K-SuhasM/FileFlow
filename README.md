# FileFlow 2026

FileFlow is a Python-based modern file organization tool that automatically sorts files into folders.

## Features

- Organize files by extension and category.
- Duplicate file detection.
- Preview file movements.
- Undo file movements.
- Light/Dark themes.
- Progress tracking.
- GitHub integration.
