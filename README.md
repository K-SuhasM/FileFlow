# FileFlow 2026

FileFlow is a modern file organization tool built using Python and CustomTkinter created by me, 

## Features

- Organize files by extension and category.
- Duplicate file detection.
- Preview file movements.
- Undo file movements.
- Light/Dark themes.
- Progress tracking.
- GitHub integration.
